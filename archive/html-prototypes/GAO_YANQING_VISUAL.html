Visual + Motion Spec

高延清样例页完整视觉与动效稿

本页是可直接预览的视觉规格展示，按首页、故事页、制作页三段组织，并包含全站音乐开关交互。用于设计评审与前端拆解。

首页 Home

目标是 5 秒内理解产品价值并进入“高延清样例”。视觉上采用深底金光、首屏高对比标题和轻量星空动效。

Hero 分层入场：标题 -> 副文案 -> CTA
样例卡片进入时上浮 reveal + 金色描边
右上角固定“开始制作”与音乐开关

故事页 Story

围绕“时间轴 + 图像注脚 + 文稿洞察”组织叙事。每屏仅一个主信息点，滚动即推进人生阶段。

时间节点进视口即高亮，上一节点降亮
回忆图像使用几何遮罩与 SVG 注脚连线
文稿区展示关键词、字数与章节密度

制作页 Create

确保“填 API -> 上传 -> 生成 -> 分享”路径清晰，避免首次用户因表单复杂而流失。

字段：Provider / API Key / Base URL / Model
上传区：肖像照 / 轨迹照 / 文稿 + 情感标签
状态流：上传中 -> 解析中 -> 生成中 -> 完成

音乐 Music

为纪念感叙事增加氛围。默认关闭，用户主动开启，状态持久化。当前接入样例音轨 `gao-theme.wav`。

按钮文案：音乐：关 / 音乐：开
开启失败提示：浏览器阻止播放
推荐音量：0.30-0.35，循环播放

滚动叙事节拍预览

示例节拍：封面建立人物印象 -> 时间线推进生平 -> 回忆图像补充细节 -> 文稿统计抽取精神关键词 -> CTA 引导制作。
