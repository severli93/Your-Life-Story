# 情绪觉察 Chat Agent — 产品架构 v1.0

> 一个帮你听见自己的 AI 对话伴侣

---

## 一、产品定位

| 维度 | 定义 |
|------|------|
| 核心价值 | 通过对话，帮助用户识别、理解、表达情绪，并建立自我叙事 |
| 目标用户 | 对自我成长感兴趣、希望更了解自己内心状态的用户 |
| 核心差异 | 不是心理咨询，不是日记App，而是一个"会倾听、有记忆、帮你看见自己"的对话空间 |
| 技术底座 | 302.ai × Claude (claude-sonnet-4-6)，Next.js 14 + TypeScript |

## 三、记忆系统设计

### 3.1 两种模式

```
                    ┌─────────────────────────────┐
                    │        记忆模式选择          │
                    │      (首次进入时引导)         │
                    └─────────────────────────────┘
                                 │
              ┌──────────────────┴──────────────────┐
              │                                     │
              ▼                                     ▼
  ┌─────────────────────────┐         ┌─────────────────────────┐
  │     🗂 生活回忆相册      │         │     📔 从现在开始记录    │
  │                         │         │                         │
  │  通过对话提取过去记忆    │         │  选择记录粒度：          │
  │  生成人生时间轴          │         │  · 日粒度（推荐）        │
  │                         │         │  · 周粒度               │
  │  导入方式：              │         │  · 月粒度               │
  │  · AI 引导对话提取       │         │  · 年粒度               │
  │  · 粘贴过去的日记/文字   │         │                         │
  └─────────────────────────┘         └─────────────────────────┘
```

### 3.2 人生回忆相册 — 对话提取流程

```
AI 引导式回忆提取:

阶段1: 时间锚定
  "你想从哪个时期开始？童年、学生时代、还是某个特别的年份？"

阶段2: 场景还原
  "那时你在哪里？身边有谁？"
  "有没有一幅画面，是那段时间最深刻的？"

阶段3: 情绪挖掘
  "当时的你，最常有的感觉是什么？"
  "有没有某种情绪，在那段时期特别熟悉？"

阶段4: 当下连接
  "现在回看那段记忆，你有什么感受？"
  "那时的经历，现在还在影响你吗？"

输出: 生成带时间戳的记忆节点，标注情绪标签，更新生命时间轴
```

### 3.3 实时记录 — 粒度说明

| 粒度 | 触发时机 | 记录内容 | AI 行为 |
|------|----------|----------|---------|
| 日粒度 | 每次对话 | 当下情绪快照 | 实时觉察 + 提问引导 |
| 周粒度 | 每周日晚 | 一周情绪回顾 | 生成周情绪摘要 + 模式识别 |
| 月粒度 | 每月末 | 情绪主题归纳 | 提炼本月情绪关键词 |
| 年粒度 | 每年末 | 年度自我叙事 | 生成情绪成长报告 |

## 五、技术架构

### 5.1 整体技术栈

```yaml
框架: Next.js 14 (App Router) + TypeScript
样式: TailwindCSS
LLM:  302.ai proxy → claude-sonnet-4-6
      API: https://api.302.ai/v1
      SDK: openai npm package (兼容接口)
存储: 本地文件系统 (JSON)，后续可接入数据库
包管理: npm
```

### 5.3 LLM 配置（延续 Design Agent 设置）

```typescript
// lib/llm.ts
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: process.env.API_302_KEY,
  baseURL: 'https://api.302.ai/v1',
});

export const DEFAULT_MODEL = 'claude-sonnet-4-6';

export async function chat(messages, systemPrompt) {
  return client.chat.completions.create({
    model: DEFAULT_MODEL,
    messages: [
      { role: 'system', content: systemPrompt },
      ...messages
    ],
    stream: true,
  });
}
```

## 七、对话路由逻辑

```
用户消息输入
    │
    ├── 检测情绪强度 (0-10)
    │   ├── 0-3: 平静/中性 → 日常陪伴对话
    │   ├── 4-6: 中度情绪 → emotion-awareness skill
    │   └── 7-10: 强烈情绪 → guided-exercise skill
    │
    ├── 检测对话意图
    │   ├── "回忆/以前/那时候" → memory-extract skill
    │   ├── "最近总是/每次都" → pattern-insight skill
    │   ├── "怎么办/帮我" → guided-exercise skill
    │   └── 其他 → 默认陪伴模式
    │
    └── 记忆粒度检查
        ├── 是否到了周回顾时间？
        ├── 是否到了月总结时间？
        └── 有无未完成的记忆节点？
```
